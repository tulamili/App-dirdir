
(2023年4月6日に、コマンド名を dirdir から dirstrata に変更し、同様に App::dirdir も App::dirstrata に変更した。
 下記の画像は、その変更の前のものである。)

このコマンドは、あるディレクトリがどんな構造をしているか把握することを目的とする。

あるディレクトリが、ファイル群として、どういうディレクトリの構造を持っているのか、調べたいときに使う。
決めたディレクトリから、i階層下にあるディレクトリでそこからさらに最大j階層下にあるディレクトリをもつような物が何個あるかを行列状に出力する。
